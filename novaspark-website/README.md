# NovaSpark Archery Website

This is the official website for NovaSpark Archery.

## Quick Deploy to Vercel

### Option 1: Direct Upload (Simplest)
1. Go to https://vercel.com/drop
2. Drag and drop this entire folder
3. Your site will be deployed instantly!

### Option 2: GitHub Integration
1. Create a new repository on GitHub
2. Upload all files in this folder to the repository
3. Go to https://vercel.com
4. Click "Add New..." → "Project"
5. Import your GitHub repository
6. Vercel will automatically deploy!

## Domain Setup

After deployment:
1. Go to Vercel project Settings → Domains
2. Add your domain: novasparkarchery.com
3. Update DNS records at Namecheap:
   - Add CNAME record: www → cname.vercel-dns.com
4. Wait 24-48 hours for propagation

## Files Included
- `index.html` - Main page
- `assets/` - JavaScript and CSS files
- `images/` - All images
- `vercel.json` - Vercel configuration

# NovaSpark 射箭官网

这是 NovaSpark 射箭品牌的官方网站。

## 快速部署到 Vercel

### 方式一：直接上传（最简单）
1. 访问 https://vercel.com/drop
2. 将整个文件夹拖拽到页面
3. 网站将立即部署！

### 方式二：GitHub 集成
1. 在 GitHub 创建新仓库
2. 将本文件夹所有文件上传到仓库
3. 访问 https://vercel.com
4. 点击 "Add New..." → "Project"
5. 导入您的 GitHub 仓库
6. Vercel 将自动部署！

## 域名设置

部署后：
1. 进入 Vercel 项目设置 → Domains
2. 添加您的域名：novasparkarchery.com
3. 在 Namecheap 更新 DNS 记录：
   - 添加 CNAME 记录：www → cname.vercel-dns.com
4. 等待 24-48 小时生效
